# 👨‍💼 HR Vision — Interactive Employee Analytics Dashboard

A modern and interactive **Employee Analytics Dashboard** built using pure **HTML, CSS, JavaScript, Chart.js, and Excel integration** for workforce intelligence, HR analytics, employee tracking, and financial insights.

Designed with a clean futuristic UI, responsive layout, dynamic filtering, and advanced chart interactions for real-time HR decision-making.

---

## 🚀 Features

* 👥 Employee Workforce Analytics
* 📊 Interactive KPI Dashboard
* 📈 Dynamic HR Visualizations
* 💰 Salary & Finance Analytics
* 🏢 Department-wise Employee Analysis
* 🚻 Gender Diversity Insights
* 📅 Hiring & Exit Trend Analysis
* 🎯 Interactive Filtering System
* 📂 Excel & CSV Upload Support
* 🔍 Employee Search Functionality
* ⚡ Real-Time Dashboard Updates
* 🎨 Modern Responsive UI

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Charts:** [Chart.js](https://www.chartjs.org?utm_source=chatgpt.com)
* **Excel Processing:** [SheetJS (XLSX)](https://sheetjs.com?utm_source=chatgpt.com)
* **Icons:** [Font Awesome](https://fontawesome.com?utm_source=chatgpt.com)
* **Styling:** Custom Responsive CSS
